# 代理式人工智能

???+ info "课程信息"

    - 讲师: Andrew Ng
    - 来源：[DeepLearning.AI](https://www.deeplearning.ai/courses/agentic-ai/)

## 课程简介

**代理式人工智能** (Agentic AI) 代表了一种利用**大语言模型** (LLM) 完成复杂任务中部分或全部步骤的软件构建新方式。与仅对提示生成单一响应不同，代理式工作流使 AI 能够规划多步流程、迭代执行，并通过反思和工具使用来优化输出。本课程将教你从零开始构建这些复杂的 AI 系统。
