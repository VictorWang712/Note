# 数字逻辑设计

???+ info "课程信息"

    - 教师：蔡铭
    - 学分：4.0
    - 学期：大一春夏

## 课程内容

- 数字表示、数字编码
- **布尔代数**和**逻辑最小化**技术
- **组合电路**设计与分析
- **时序电路**设计与分析，时序电路的时序分析
- **结构化**和行为建模的逻辑设计
- 可编程逻辑器件（**PLD**）和**存储器**

## 分数构成

- 实验：30%
- 项目：10%
    - 需提交源代码、源项目和技术报告
    - 在期末考试前提交
- 小测：20%
    - 四至五次，不会有事先通知
    - 题目源自课本和课后习题
- 期末考试：40%
